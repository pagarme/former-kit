#### **Examples** ####

Default
```jsx
<Avatar alt="username" />
```

With default size and custom photo
```jsx
<Avatar
  alt="username"
  photo="https://i.imgur.com/V9mgrCp.jpg"
/>
```

With custom size and photo
```jsx
<Avatar
  alt="username"
  photo="https://i.imgur.com/V9mgrCp.jpg"
  size={50}
/>
```
